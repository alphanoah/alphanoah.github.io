HomeAssistant 将米家设备接入 HomeKit 控制

安装 HACS

方法1（代理走不了行不通）

  • 文档
  • 命令说明
    • wget -O - https://get.hacs.xyz | bash -
    • -O的含义是把后面网址下载后，改成一个指定的名称，如果后面没有跟着一个名字，而是“-”，则表示将下载后的内容输出到标准输出，也就是输出到屏幕上。
    • 然后通过|，传递给bash执行
    • 减号"-“就是代表标准输出/标准输入, 视命令而定. “-“代替stdin和stdout的用法,stdin就是标准输入,stdout就是标准。 http://http://raw.staticdn.net/hacs/get/main/get GitHub raw 域名并非 http://github.com 而是 http://raw.githubusercontent.com，上方的 GitHub 加速如果不能加速这个域名，那么可以使用 Static CDN 提供的反代服务。

将 http://raw.githubusercontent.com 替换为 http://raw.staticdn.net 即可加速。 —脚本内需要外网不可行

方法2 离线安装

  1. release包下载
  2. 在homeassistant中点击Supervisor安装Samba share目录共享插件
  3. 在Home Assistant根目录创建custom_components与www文件
  4. 安装
    1. 进入Home Assistant主界面
    2. 点击左下角配置
    3. 进入设备与服务
    4. 点击右下角 ➕添加集成
    5. 搜索hacs并点击下载
    6. 四项全打钩（否则无法进入）
    7. 授权GitHub
    8. 完成
  5. 米家设备接入homeassistant
    1. 点击导航栏「HACS」>「集成」，选择「浏览并下载存储库」。
    2. 搜索xiaomi 并安装Xiaomi MIoT，网络问题可以离线安装！！！
    3. 重启：配置->系统->重新启动
    4. 配置-集成-添加集成-搜索xiaomi-安装xiaomi MIoT
    5. 输入 米家 登陆账号及密码
    6. 选择想接入HomeKit的米家设备（建议一个一个添加尝试）
  6. homeassistant设备接入HomeKit
    1. 配置-集成-添加集成-搜索HomeKit并安装
    2. 点击提交
    3. 点击左侧导航栏 通知
    4. 手机打开家庭APP 添加设备扫描此二维码
