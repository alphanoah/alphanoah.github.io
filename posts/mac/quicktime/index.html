解决 Mac(Apple Silicon) 录屏带系统声音

mind-map

背景

  • 由于 Quicktime Player 屏幕录制无法录制系统自带声音
  • soundflower 可能没法和 Apple Silicon 适配，所以使用 Blackhole

步骤

  1. 安装 Blackhole （使用brew方式）

    brew install blackhole-2ch

  2. 音频 Midi 设置

    1. 打开 音频 Midi 设置
    2. 在音频设备的左下角单击 + 图标。选择 创建多输出设备
    3. 选择输出设备，在刚创建的设备中，选择： MacBook Pro 扬声器 在前，blackhole 在后，并启用后者的 漂移校正 （具体参考）。

      必须启用内置输出（MacBook Pro 扬声器）并将其列为多输出中的顶部设备，否则音频和视频无法同时播放。

  3. 将音频输出设置为多输出设备

    • 右键单击您的多输出设备，然后选择 将此设备用于声音输出 。

  4. 恢复音量调整

    • 在 设置 -> 声音 -> 输出与输入 选择 MacBook Pro 扬声器 即可。

      为什么要这样设置，因为 macOS 不支持更改多输出设备的音量，如果觉得麻烦，可以按插件的建议可以在音频 MIDI 设置中设置单个设备的音量。
